# Penugasan-Arduino-NCC
## Ultrasonic Sensor Glasses

Untuk penugasann ketiga sekaligus yang terakhir ini, saya memilih project IoT yaitu Ultrasonic Sensor Glasses. Ultrasonic glasses ini adalah sebuah kacamata yang dimodifikasi dengan sensor ultrasonic untuk mendeteksi hal hal yang terdapat di depan pengguna. Kacamata ini cocok digunakan untuk seorang tuna netra untuk membantu berjalan dan mendeteksi keadaan sekitar.
